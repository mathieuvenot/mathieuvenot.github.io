D6W_ MACHINE VISION

18/02/2013 14:00 Categories: Other Work / ENSAPM

// ART_CHITECT 0.1A (With CLEMENT GOSSELIN & PAUL POINET)







Iterative process which generates a building, a city and a landscape from a color code (RGB) of an image. RoboRealm (machine vision software) analyzes the 2D image and sends the information to Grasshopper, which applies 3D transformations.
